# Shop Sales Analysis Dashboard

## Project Overview

This project analyzes one week of real-world shop sales data using Microsoft Excel. The objective is to identify sales trends, profitable products, and business opportunities through data analysis and dashboard visualization.

## Tools Used

* Microsoft Excel
* Excel Tables
* Pivot Tables
* Pivot Charts
* Dashboard Design

## Dashboard Preview

![Dashboard](data/screenshots/Shop%20Dashboard.png)

## Analysis Performed

### Daily Profit Analysis

* Calculated daily profit trends.
* Identified the most profitable sales day.

### Product Profit Analysis

* Analyzed profit generated by each product.
* Identified top-performing products.

### Product Quantity Analysis

* Measured product demand based on quantity sold.
* Determined best-selling products.

## Key Metrics

* Total Weekly Profit: ₹3,685
* Best Sales Day: 25-05-2026
* Top Product by Profit: Tank 1000 Ltr
* Average Daily Profit: ₹526.40

## Key Insights

* Total weekly profit reached ₹3,685.
* Tank 1000 Ltr generated the highest profit.
* 25-05-2026 was the most profitable day.
* Profit showed a gradual decline toward the end of the week.
