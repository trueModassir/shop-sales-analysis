# Data Folder

This folder contains the project dataset, Excel dashboard workbook, and supporting project assets.

## Contents

### Shop_Sales_Analysis_Dashboard.xlsx

Contains:

* Raw sales data
* Pivot table analysis
* Pivot charts
* Interactive dashboard
* Business insights

### screenshots/

Contains project screenshots used in the main repository documentation.

## Data Description

The dataset contains one week of shop sales records with the following fields:

* Date
* Product
* Quantity Sold
* Selling Price
* Cost Price
* Profit

The data is used to analyze product performance, profitability, and weekly sales trends.
